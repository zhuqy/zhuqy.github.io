HTML的结构与标签

名人说 基础是人类进步的阶梯 所以我们要注意脚下

今天和大家谈一谈 HTML的结构与标签 (=￣ω￣=)

1 - HTML 结构

<!doctype html>//头文档声明，表示使用的是h5语言
<html>
<head>
    <meta charset="utf-8"/>//编码格式
    <title></title>
    <link rel="stylesheet" type="text/css" href="css/index.css"/>//css外部引入
</head>
<body>
    <div></div>
</body>
</html>

2 - HTML 标签

  • html 是根标签，用作浏览器识别使用。

  • head 放入html的元数据，用于机器阅读。(title script link style meta base)

  • body 是显示网页内容。

HTML 标签可分为三种：

  1. 块级标签：默认继承父级宽高，也可以自设宽高，margin，padding。
如：< body >、< div >、< h1 > ~ < h6 >、< p >、< form >、< ol >、< ul >、< dl >、< table >

  1. 行级标签：默认没有宽高，由内容撑开，没有margin-top/bottom,padding-top/bottom,成行状排列。

    如：< a >、< span >、< img / >

  2. 无行块之分：< br / >

可用display属性更改：

display:none;//不显示

display:inline;//显示为行级标签

display:block;//显示为块级标签

display:inline-block;//设置拥有块、行级属性，既可自设宽高，又可成行状排列。

HTML 标签虽然对大小写不敏感，但为了使用规范推荐使用小写。
