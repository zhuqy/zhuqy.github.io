HTML属性

名人说 基础是人类进步的阶梯 所以我们要注意脚下

今天和大家谈一谈 HTML的属性 (=￣ω￣=)

HTML 元素

  1. 标题标签：< h1 > < / h1 >

    标题一般用来呈现文档结构，不能乱用。

    由标题标签大到小为 < h1> ~ < h6 > 。

    有默认的margin值，默认字体加粗。

  2. 区域标签：< div > < / div >

    起容器作用。默认继承父级宽高。

  3. 段落标签：< p > < / p >

    起包裹作用。默认继承父级宽高，有默认的margin值。

  4. 链接标签：< a href=”url” > < / a >

    属性有：href 必须存在 | title 起提示作用 | target 设定是否在新页面打开链接

    < a > 标签有默认的字体颜色、下划线及下划线颜色，宽度由内容填充。其中href的值为指定链接的地址，不能为空否则属性失效，可用#代替，#可与id配合形成锚点。

  5. 图像标签：< img / >

    属性有：src=”图像的位置” | alt=”指定图像的描述性文本，当图像不可见时（下载不成功时），可看到该属性指定的文本” | title=”提供在图像可见时对图像的描述(鼠标滑过图片时显示的文本)”

    作为特殊的行级标签可直接设置宽高。在部分浏览器中有边框属性，需要去除以免影响浏览效果。

  6. 列表标签

    有序列表
<ol>
    <li></li>
    <li></li>
</ol>

无序列表

<ul>
    <li></li>
    <li></li>
</ul>

字典列表

<dl>
    <dt></dt>
    <dd></dd>
</dl>

    列表标签需要配套使用，中间不能嵌套其他标签。有默认的padding值，列表标记list-style-type值，使用时建议清除。

  1. 强调标签：< em > < / em > 斜体 | < strong > < / strong > 加粗

    < em > 表示强调，< strong > 表示更强烈的强调。

  2. 短文本引用：< q > < / q >
    长文本引用：< blockquote > < / blockquote >

  3. 换行标签：< br / >

  4. 分割线标签：< hr / >

  5. 在页面中加入地址信息：< address > < / address >

    <address>上海市松江区丽德创业园</address> //样式为斜体

  6. 显示一行代码：< code > < / code >

    预格式化文本：< pre > < / pre >

    要插入多行代码时不能使用< code >标签，可以使用< pre > 标签。

  7. 表格标签：< table > < / table >

  • < table > < / table >

  • < tbody > < / tbody > tbody这个标签基本上不怎么用了

  • < tr > < / tr > 表格的一行

  • < th > < / th > 表格的头部的一个单元格，表格表头

  • < td > < / td > 表格的一个单元格

  • < caption > < / caption > 表格标题

  1. 表单标签：
  • < form > < / form >

action属性 输入的数据被传送到的地址,比如一个PHP页面(test.php) | method属性 数据传送的方式（get/post）

－ 输入框 < input / >

  • 当type=”text”时，输入框为文本输入框

  • 当type=”password”时，输入框为密码输入框

  • 当type=”radio”时，控件为单选框

  • 当type=”checkbox”时，控件为复选框

  • 当type=”submit”时，控件为提交

  • 当type=”reset”时，控件为重置

  • 当type=”hidden”时，控件为隐藏

  • 当type=”button”时，控件为按钮

    name 为文本框命名或控件命名，以备后台程序ASP 、PHP使用 | value 为文本输入框设置默认值(一般起到提示作用)或提交数据到服务器的值（后台程序PHP使用） | checked 当设置checked 或 checked=”checked”时，该选项被默认选中，与 或 配合使用。

JS之数组操作

数组的操作

预处理：var arr = [1,2,3,4,5];

1. join( )

  • 用于把数组中的所有元素放入一个字符串。元素通过指定的分隔符进行分隔，参数就是指定的分隔符。

  • 当分隔符为无，可将把数组直接转化为字符串。

    console.log(arr.join("|"));//1|2|3|4|5;

JS之字符串操作

字符串的操作

预处理：var str = “hello world”;

1. indexOf()

  • 查询相应字符串及其相应的位置。返回当前字符的下标值，为0~n。

  • 所以常被用来判断字符串是否有某些数值。

    console.log(str.indexOf("o"));//4;
console.log(str.indexOf("haha"));//-1;不存在的值 一律返回-1
console.log(str.indexOf("o",5));//7;从下标值为5的位置开始寻找
console.log(str.lastIndexOf("0",10));//7;从字符串尾部开始寻找
